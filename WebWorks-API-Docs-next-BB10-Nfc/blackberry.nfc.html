API Reference

NFC

The Nfc object contains functions for handling and preparing ndef nfc Tags.

The functionality in this object allows you to emulate read and write smart tags.

static void blackberry.nfc.addMimeTypeListener (mimeType : String, onTagDetected: function(type : String, tag : Object), onSuccess : function, onFailure : function)


Registers an event listener for tags matching the specified MIME type.
This event occures when a TNF_MIME_MEDIA tag with the specified MIME type is read.


Supported Platforms
 - BlackBerry 10
 - Ripple Emulator


Parameters
mimeType The MIME type filter for messages.
onTagDetected The callback called when a tag is detected with the specified MIME type.

type: The event type.
tag: The Ndef tag. *NOTE This objects properties are platform dependent.
onSuccess Optional callback function that is invoked when the listener is added.
onFailure Optional callback function that is invoked when there was an error in adding the listener.
